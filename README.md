# 拼豆像素画生成器

一个专业的在线像素画生成工具，专为拼豆创作设计。将您的图片快速转换为适合拼豆制作的像素画。

## 功能特点

- 图片上传：支持拖拽上传或点击上传，兼容 JPG、PNG、GIF 等常见格式
- 像素化处理：可调节像素块大小（2px - 50px），从精细到粗糙自由选择
- 色彩调整：支持亮度、对比度、饱和度调节，优化色彩效果
- 拼豆色板：内置80种拼豆标准色号，自动匹配最接近的颜色
- 色号显示：可在每个像素上显示对应的拼豆色号，方便制作
- 色号图例：显示当前使用的所有拼豆色号及其名称
- 实时预览：参数调整即时显示效果，原图与像素画对比展示
- 网格显示：可选显示网格线，方便拼豆制作时参考
- 高清导出：支持 PNG 格式导出，可选择 1x、2x、3x、5x 倍放大
- 响应式设计：适配各种屏幕尺寸，移动端也能流畅使用
- 浏览器兼容：支持主流浏览器（Chrome、Firefox、Safari、Edge）

## 使用方法

### 1. 上传图片
- 点击上传区域选择图片，或直接将图片拖拽到上传区域
- 支持的图片格式：JPG、PNG、GIF

### 2. 调整参数
- **像素大小**：控制像素块的尺寸，数值越小越精细，数值越大越粗糙
- **亮度**：调整图片的明暗程度
- **对比度**：调整图片的色彩对比
- **饱和度**：调整图片的色彩鲜艳程度
- **使用拼豆色板**：勾选后，图片颜色将自动匹配到最接近的拼豆标准色
- **显示色号**：勾选后，每个像素块上将显示对应的色号（拼豆色号或十六进制颜色代码）
- **显示网格线**：勾选后，显示像素块边界
- **显示原图对比**：勾选后，同时显示原图和像素化效果

### 3. 预览效果
- 左侧显示原图，右侧显示像素化效果
- 调整参数时实时更新预览
- 可勾选"显示网格线"查看像素块边界

### 4. 导出图片
- 选择导出尺寸（原始尺寸或放大倍数）
- 点击"导出图片"按钮
- 图片将以 PNG 格式下载到本地

## 快捷键

- `Ctrl + S` / `Cmd + S`：快速导出图片
- `Ctrl + R` / `Cmd + R`：重置所有参数

## 技术特点

- 纯前端实现，无需服务器
- 使用 Canvas API 进行图像处理
- 采用最近邻插值算法实现像素化
- 防抖优化，提升性能
- 响应式布局，适配各种设备

## 使用建议

1. **选择合适的图片**：建议使用色彩丰富、轮廓清晰的图片
2. **调整像素大小**：根据拼豆板大小选择合适的像素尺寸
3. **使用拼豆色板**：勾选"使用拼豆色板"后，颜色会自动匹配到标准拼豆色
4. **显示色号**：勾选"显示色号"可以直观看到每个像素对应的色号，方便制作
5. **查看色号图例**：使用拼豆色板时，下方会显示所有使用的色号和颜色名称
6. **优化色彩**：适当调整亮度、对比度和饱和度，使色彩更鲜艳
7. **导出放大**：建议使用 3x 或 5x 放大导出，便于查看细节
8. **显示网格**：制作时可开启网格线，帮助定位每个像素块

## 常见问题

**Q: 为什么上传大图片后处理速度慢？**
A: 系统会自动将大图片缩放到合适尺寸（最大 800px），以保证处理速度。

**Q: 导出的图片不够清晰？**
A: 尝试增加导出尺寸倍数，或减小像素块大小。

**Q: 如何获得更好的色彩效果？**
A: 适当提高对比度和饱和度，使色彩更加鲜艳。

**Q: 拼豆色板是什么？**
A: 拼豆色板包含了80种标准拼豆颜色，勾选"使用拼豆色板"后，图片颜色会自动匹配到最接近的标准色。

**Q: 色号显示的是什么？**
A: 当使用拼豆色板时，显示的是拼豆色号（如 01、02 等）；不使用拼豆色板时，显示的是十六进制颜色代码（如 #FF0000）。

**Q: 如何查看使用了哪些拼豆色号？**
A: 勾选"使用拼豆色板"后，预览区域下方会显示所有使用的色号图例，包含色号、颜色名称和色块预览。

**Q: 可以导出其他格式吗？**
A: 目前仅支持 PNG 格式，这是最适合像素画的格式。
